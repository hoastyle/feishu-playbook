# 快速开始

欢迎使用 feishu-playbook！这是一个帮助研发团队自动化飞书文档管理的项目。

## 第一步：了解 AI Skills

AI Skills 是本项目的核心，它们让 AI Agent（如 Claude、Cursor、Windsurf）能够快速掌握飞书文档操作。

## 第二步：选择你的使用场景

### 场景 1：日常文档上传

你有一个 Markdown 文档，想上传到飞书。

**对话示例**：
```
你：把 README.md 上传到飞书
AI：好的，我来帮你上传 README.md 到飞书...
```

### 场景 2：批量迁移

你有一个文件夹的文档，想批量迁移到飞书。

**对话示例**：
```
你：把 docs/api/ 文件夹的所有文档上传到飞书 Wiki
AI：好的，我将批量上传这些文档...
```

### 场景 3：AI 辅助编辑

你想让 AI 帮助优化飞书文档的格式。

**对话示例**：
```
你：优化飞书文档的格式
AI：好的，我来帮你优化文档格式...
```

## 第三步：配置环境
